Create a New Segment

To Create a Segment, follow these steps:

  1. Login to Video Cloud.
  2. In the primary navigation, click Analytics.
  3. In the Custom section, click Segments.
  4. To create a new segment, click Create .
  5. Give your segment a name.
  6. In User Filters, click + Add user filter . User Filters allow us to segment based on different subscription based attributes.

    You can also select an Engagement Status.

    To learn more about what engagement status means, see Audience Insights Glossary

    To add more filters, click + Add user filter . You can add many filters as you want.

  7. Click Add
  8. In Content Filters, click + Add content filter . Content Filters allow us to segment based on other content attributes.
    • a. Choose a period between 3 to 2 years
    • b. and c. Choose any: watched, watched-happy, watched-unhappy or not watched
    • d. Select content.
  9. Click Add.
  10. You can have your segments either Active or Inactive.
  11. Click Preview so you can see the total rows. And then, click Create.

    It takes a few minutes to create the segment. In the meantime, you can see In progress in the status. When the segment is created, you will receive a notification. Once it has been created, you can see active or Inactive status.

    12. Notes:
    • The ID segments are for identifying each segment
    • The time it takes to create a segment depends on how big the reports are, some reports are created in seconds, and others can take minutes. You can see a notification when the report is made, or you can also click the refresh button
    • You can download the segment, or you can also push it into the integrations through Video Cloud
    • Once you create a segment, it automatically updates your integrations. However, you can still use segments without having to use those integrations. So you do not need to establish the integrations to the account. You would still be available to create segments and download reports
    • You can only have ten active going segments; creating a new one is impossible after that number, even if you want to set it up as inactive
    • You can go back to filters and make updates, and it will create a new report.
