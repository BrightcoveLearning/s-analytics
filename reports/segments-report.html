Create a New Segment

To Create a Segment, follow these steps:

  1. Login to Video Cloud.
  2. In the primary navigation, click Analytics.
  3. In the Custom section, click Segments.
  4. To create a new segment, click Create .
  5. Give your segment a name.
  6. In User Filters, click + Add user filter . User Filters allow to segment based on different subscription based attributes.

    You can also select an Engagement Status.

    To learn more about what engagement status, see Audience Insights Glossary

    To add more filters, click + Add user filter . You can add many filters as you want.

  7. Click Add
  8. For Content Filters, click + Add content filter . Content Filters allow us to segment based on other content attributes.
    • a. Choose a period between 3 days to 2 years
    • b. and c. Choose any: watched, watched-happy, watched-unhappy or not watched
    • d. Select content.
  9. You can add additional filters. Click Add after each selection, to ensure your selections are saved.
  10. You can have your segments either Active or Inactive.
    Note: segments are set to Active by default at creation.
  11. Click Preview to see the total rows. And then, click Create.

    It takes a few minutes to create the segment. The status changes to In progress . When the segment is created, you will receive a notification on the screen. Once it has been created, you can see active or Inactive status.

    12. Notes:
    • Editing new filters will override any reports generated with the previous filters
    • The Segment IDs are unique to each segment and allow you to identify each segment
    • The time it takes to create a segment could vary depending on the size of segment being generated. You will see a notification, when the segment is created. You can also click the refresh button to get the latest status
    • A user can be associated with multiple Segment IDs , and will have each of these segment IDs tagged to user records that can be synchronized with CRM or MAP systems, or be downloaded as a report
    • Creating a segment is not dependent on establishing integrations with MAP/CRM Systems. Segments can be created and downloaded as a report
    • Once you create a segment and set it to active status, it automatically updates your integrations. However, you can still use segments without having to use those integrations. So you do not need to establish the integrations to the account. You would still be available to create segments and download reports
    • There is a limit of 10 active segments allowed at a time. If you need to create more segments, set one of the segments to inactive to create a new segment
    • Once a report is generated, you will have Seven days to view it before it is deleted. The segment will remain active, however, the latest report needs to be regenerated.
